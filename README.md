# 安装

```
$ npm install -g linec / cnpm install -g linec

```

# 使用

基础用法
```shell
$ linec
```

导出到html
```shell
$ linec -h
```

# 功能

- 输出空行，实际行数，总行数
- 支持400+语言
- 显示遍历速度
- 显示多种颜色
- 支持导出html
